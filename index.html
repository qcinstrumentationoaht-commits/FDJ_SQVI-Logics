Logic 1 – ZFDJ_QI-SMP

SQVIQMFinal

Title

Quality Inspection Stock Movement Pending List

Data Source

Table Join

Tables & Join Strategy

  • MCHB → MARA (INNER JOIN on MATNR)
  • MCHB → MAKT (INNER JOIN on MATNR)
  • MCHB → QALS (INNER JOIN on WERKS, MATNR, CHARG, LGORT)
  • QALS → AUFK (LEFT OUTER JOIN on AUFNR)

Selection Conditions

  • QI Stock must exist (CINSM ≠ 0)
  • Stock posting not completed (STAT34 ≠ X)
  • Usage decision pending (STAT35 ≠ X)
  • Inspection type excluded: 03
  • Plant, batch, order category – optional

Output

  • Plant, Material, Description, Material Type
  • Storage Location, Batch, QI Quantity
  • Created / Changed By & Dates
  • Order Number, Order Dates

Sort Logic

Sorted by Last Changed By (AENAM) in ascending order

Business Intent

Provides a single operational view of QI stock awaiting quality or posting action, enabling accountability, follow-up, and audit readiness.
