Logic 1 – ZFDJ_QI-SMP

Title

Quality Inspection Stock Movement Pending List

Data Source

Table Join

Tables Used & Join Logic

  • MCHB → MARA (INNER JOIN on MATNR)
  • MCHB → MAKT (INNER JOIN on MATNR)
  • MCHB → QALS (INNER JOIN on WERKS, MATNR, CHARG, LGORT)
  • QALS → AUFK (LEFT OUTER JOIN on AUFNR)

Selection Criteria

  • Plant (MCHB-WERKS) – Optional
  • Stock in QI (MCHB-CINSM) ≠ 0
  • Order Category (AUFK-AUTYP) – Optional
  • Batch Number (MCHB-CHARG) – Optional
  • Stock Postings Completed (QALS-STAT34) ≠ X
  • Usage Decision Made (QALS-STAT35) ≠ X
  • Inspection Type (QALS-ART) ≠ 03

Output Fields

  • Plant – MCHB-WERKS
  • Material Number – MCHB-MATNR
  • Material Description – MAKT-MAKTX
  • Material Type – MARA-MTART
  • Storage Location – MCHB-LGORT
  • Batch Number – MCHB-CHARG
  • Stock in Quality Inspection – MCHB-CINSM
  • Created By / On – MCHB-ERNAM, MCHB-ERSDA
  • Changed By / On – MCHB-AENAM, MCHB-LAEDA
  • Order Number – QALS-AUFNR
  • Order Created On – AUFK-ERDAT
  • Technical Completion Date – AUFK-IDAT2

Sort Sequence

MCHB-AENAM (Name of Person Who Changed Object) – Ascending

Business Purpose

This logic provides a consolidated view of batch-wise QI stock where inspection activities are still pending, enabling quality and operations teams to take timely action.
